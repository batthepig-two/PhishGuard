# 🛡️ PhishGuard

Free phishing link checker — works on iPhone, Android & desktop.  
The extension download is built right into the website — no extra files needed.

**Live site:** https://YOUR-USERNAME.github.io/phishguard

---

## Deploy to GitHub Pages (2 minutes)

1. Create a new **public** repo at github.com/new named `phishguard`
2. Upload `index.html` and `README.md` — that's it!
3. Go to **Settings → Pages → Branch: main / root → Save**
4. Visit `https://YOUR-USERNAME.github.io/phishguard` in ~60 seconds

> Visitors can download the Chrome/Firefox extension directly from the site.  
> No extra files needed — the extension zip is built in the browser on download.

---

## Customise

- Edit `index.html` to change design or detection rules
- Replace `YOUR-USERNAME` in the footer with your GitHub username
- Optional: add a Google Safe Browsing API key in the detection code for stronger results
